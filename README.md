# A2 Minor Module Coursework — Medical Imaging

MPhil in Data Intensive Science, Lent 2026
Taught by Dr. A. Biguri, Dr. J. Kaggie, Dr. L. Escudero Sánchez

## Overview

This repository contains the code and written answers for the A2 Minor Module Coursework on Data Science Applications to Medical Imaging.

The analysis notebook is `solution.ipynb` and the written report is in `report/`.

The coursework covers three modules:
- **Module 1:** Tomographic Reconstruction (CT)
- **Module 2:** MRI Image Denoising
- **Module 3:** Image Segmentation Methods (literature review)

## Installation

Clone the repository, create a virtual environment, and install the package:

```bash
git clone https://gitlab.developers.cam.ac.uk/phy/data-intensive-science-mphil/assessments/a2_coursework/eaz21.git
cd eaz21
python -m venv venv
source venv/bin/activate
pip install -e .
```

## Running the Tests

```bash
pytest -s tests/*
```

## Building the Documentation

```bash
cd docs
make clean
make html
open _build/html/index.html
```

## Docker

To build and run the package in an isolated container:

```bash
./scripts/docker-start.sh
```

This builds the Docker image, installs the package, and runs the full test suite. To open an interactive Python shell inside the container:

```bash
docker run -it --rm medical-imaging python
```
