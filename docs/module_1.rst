Module 1: CT Reconstruction
===========================

This module implements CT image reconstruction from sinograms, including
Filtered Back-Projection (FBP), gradient descent (SIRT), and Ordered Subsets
SART (OS-SART), along with noise simulation and visualisation utilities.
