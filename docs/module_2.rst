Module 2: MRI Denoising
=======================

This module implements an MRI processing pipeline including k-space loading
and visualisation, inverse FFT to image space, root-sum-of-squares coil
combination, and denoising with Gaussian, bilateral, wavelet, and Butterworth
filters.
